HTML Structure – गहराई से समझें

1. <!DOCTYPE html>

HTML5 declaration होता है, जिससे ब्राउज़र को यह पता चलता है कि यह HTML5 document है।

2. <html lang="hi">

यह पूरे HTML डॉक्युमेंट का root tag होता है, जिसमें दो main हिस्से होते हैं: <head> और <body>।

3. <head>

इसमें meta जानकारी होती है जैसे title, charset, external CSS या JS फाइलें:

<meta charset="UTF-8">
<title>My Page</title>
<link rel="stylesheet" href="style.css">

4. <body>

जो content user को दिखता है वह body के अंदर होता है जैसे headings, paragraphs, images, lists, आदि।

5. Semantic Tags

<header>, <main>, <section>, <article>, <nav>, <footer> जैसे tags SEO और accessibility में मदद करते हैं।

6. Forms, Lists, Tables

Data input के लिए forms और structured content के लिए tables/lists का उपयोग:

<form><input type="text"></form>
<ul><li>Item</li></ul>
<table><tr><td>Data</td></tr></table>

7. Final Structure Example

<!DOCTYPE html>
<html lang="hi">
  <head>...</head>
  <body>
    <header>...</header>
    <main>...</main>
    <footer>...</footer>
  </body>
</html>
