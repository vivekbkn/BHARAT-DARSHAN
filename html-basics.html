HTML Basics – 10 जरूरी Concepts

1. HTML क्या है?

HTML (HyperText Markup Language) वेब पेज का ढांचा है जो content को structure देता है।

2. HTML Structure

HTML डॉक्युमेंट में DOCTYPE, html, head और body जैसे हिस्से होते हैं जो पेज को organize करते हैं।

3. Elements और Tags

HTML tags से elements बनते हैं जो content का structure और meaning बताते हैं।

4. Attributes

Attributes HTML tags में extra जानकारी जोड़ते हैं, जैसे class, id, src, href आदि।

5. Head vs Body

<head> में meta info, CSS आदि होता है, और <body> में वह content जो user को दिखाई देता है।

6. Semantic HTML

Semantic tags जैसे <article>, <section>, <nav> content का सही अर्थ बताने में मदद करते हैं।

7. Entities & Symbols

Special characters जैसे <, >, ₹, © आदि HTML entities के ज़रिए लिखे जाते हैं।

8. HTML5 के नए Tags

HTML5 में नए semantic tags शामिल हुए जैसे <header>, <footer>, <main>, <aside>, <figure> आदि।

9. Best Practices

Clean code, indentation, semantic tags का प्रयोग और accessible structure को follow करना चाहिए।

10. Practice & Mini Project

हर concept पर अभ्यास ज़रूरी है — छोटे HTML projects से समझ और अनुभव दोनों बढ़ते हैं।
