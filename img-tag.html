<img> टैग — डिटेल मार्गदर्शिका

<img> टैग HTML में चित्र दिखाने के लिए उपयोग होता है। यह self‑closing टैग है, यानि अलग से </img> लिखना नहीं पड़ता।

सिंटैक्स

<img src="image_url" alt="image_description" width="500" height="300">

प्रमुख attributes

  • src (source): चित्र का पथ या URL।
  • alt (alternative text): वैकल्पिक वर्णन; SEO व accessibility हेतु।
  • width / height: पिक्सल या प्रतिशत में आकार।
  • title: hover पर दिखने वाला सूचना‑पॉप‑अप टेक्स्ट।
  • loading: lazy या eager — लोड प्राथमिकता।
  • usemap: इमेज मैप हेतु; क्लिक योग्य क्षेत्र तय करता है।
  • class / id: CSS व JS चयन के लिए।
  • style: inline CSS।

उदाहरण 1 — बुनियादी

<img src="sunset.jpg" alt="A beautiful sunset" width="600" height="400">

उदाहरण 2 — Hover text

<img src="mountain.jpg" alt="Beautiful mountain" title="Click for more details" width="500" height="300">

उदाहरण 3 — Lazy Loading

<img src="beach.jpg" alt="Relaxing beach" loading="lazy">

उदाहरण 4 — Image Map

<img src="map.jpg" usemap="#region1" alt="Map" width="500" height="300">
<map name="region1">
  <area shape="rect" coords="34,44,270,350" alt="Region 1" href="https://example.com">
</map>

Best Practices

  • alt हमेशा लिखें — SEO व screen reader सहायता।
  • इमेज ऑप्टिमाइज़ (कंप्रेस) करें — तेज़ पेज लोड।
  • Responsive CSS: img{width:100%;height:auto;}
  • सही फ़ॉर्मेट चुनें — JPG, PNG, SVG, WebP आदि।

निष्कर्ष

<img> टैग सरल लेकिन महत्त्वपूर्ण है। src, alt, width/height आदि के सही प्रयोग से वेबसाइट की चित्र प्रदर्शनी, परफ़ॉर्मेंस व एक्सेसिबिलिटी सुधरती है।
