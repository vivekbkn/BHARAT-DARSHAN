Introduction

HTML (HyperText Markup Language) is the standard markup language used to create web pages.

HTML Structure

Basic structure includes doctype, html, head, and body tags.

Common Tags

Some common tags are: <h1>, <p>, <img>, <a>, etc.

Attributes

Attributes provide additional information about elements, like href in <a> tag.

Lists

HTML supports ordered (<ol>) and unordered (<ul>) lists.

Tables

Tables are created using <table>, <tr>, <td>, and <th>.

Forms

Forms are used to collect user input. Common elements are <input>, <textarea>, <button>.
